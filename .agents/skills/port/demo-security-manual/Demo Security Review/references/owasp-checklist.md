# OWASP-oriented checklist (abbreviated)

- [ ] Injection (SQL, command, template)
- [ ] Broken authentication / session handling
- [ ] Sensitive data exposure (logs, APIs)
- [ ] XML/entity parsing where applicable
- [ ] Broken access control on Port actions
- [ ] Security misconfiguration (defaults, CORS)
- [ ] XSS in custom widgets / markdown
- [ ] Insecure deserialization
- [ ] Known vulnerable dependencies
- [ ] Insufficient logging & monitoring
